# yax-hackernews

HackerNews clone built with [Yax](https://github.com/d-band/yax), based on [dva-hackernews](https://github.com/dvajs/dva-hackernews).

## Run Locally

Install dependencies.

```bash
$ npm install
```

Start server.

```bash
$ npm start
```

Open app in browser.

```bash
$ open http://localhost:8000/
```
